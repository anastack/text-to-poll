## Quiz Poster Bot (Telegram)

Бот принимает текст в формате “вопрос + варианты ответа”, превращает его в **Telegram Poll** в режиме **викторина** и автоматически публикует в привязанный канал.

Если перед вопросом была картинка (фото), бот публикует **картинку в канал**, а затем **викторину**.

### Требования
- Python 3.10+
- Бот добавлен в канал как **админ** (права на публикацию сообщений и опросов).

### Быстрый старт (Windows / PowerShell)
1) Установите зависимости (в корне репозитория уже есть `requirements.txt`):

```bash
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2) Создайте файл окружения для нового бота:

```bash
copy quiz_poster_bot\.env.example quiz_poster_bot\.env
notepad quiz_poster_bot\.env
```

3) Запуск:

```bash
python -m quiz_poster_bot.quizbot.main
```

### Формат сообщения
В личке боту отправьте:

- 1-я строка: **вопрос**
- следующие строки: **варианты** (2–10 шт)
- правильный вариант пометьте `*` или `+` в начале строки

Пример:

```text
Сколько будет 2+2?
*4
3
5
22
```

### Как работает фото
- Если вы отправите **фото с подписью** в формате выше — бот запостит фото в канал и сразу следом викторину.
- Если вы отправите **фото без подписи**, а потом текст-вопрос — бот привяжет фото к следующему вопросу (в пределах таймаута).
