## Quiz Poster Bot (Telegram)

Бот принимает текст в формате "вопрос + варианты ответов", превращает его в Telegram Poll в режиме викторины и публикует в выбранный канал.

Можно отправлять один вопрос, блок вопросов на тему или поставить блок в очередь на публикацию через указанное время.

После `/start` бот показывает inline-кнопки: создание блока, отложенная отправка, очередь, смена канала и помощь.

### Требования

- Python 3.10+
- Бот добавлен в канал как админ с правом публикации сообщений и опросов.

### Быстрый старт (Windows / PowerShell)

```bash
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy quiz_poster_bot\.env.example quiz_poster_bot\.env
notepad quiz_poster_bot\.env
python -m quiz_poster_bot.quizbot.main
```

### Настройки

`BOT_TOKEN` обязателен.

`TARGET_CHANNEL_ID` можно указать в `.env`, например `-1001234567890` или `@your_public_channel`. Если оставить пустым, бот попросит канал в чате. Канал также можно заменить командой `/channel`.

### Один вопрос

Отправьте боту:

```text
Сколько будет 2+2?
*4
3
5
22
```

Правильный вариант помечается `*` или `+`. Если правильных вариантов несколько, бот отправит обычный опрос с множественным выбором.

### Блок вопросов на тему

```text
/block История
Кто был первым президентом США?
*Джордж Вашингтон
Авраам Линкольн
Томас Джефферсон

В каком году началась Вторая мировая война?
*1939
1941
1945
```

Вопросы можно разделять пустой строкой или строкой `---`. Вместо темы в команде можно написать первой строкой `Тема: История`.

### Отложенная отправка

```text
/schedule 10m История
Кто был первым президентом США?
*Джордж Вашингтон
Авраам Линкольн

В каком году началась Вторая мировая война?
*1939
1941
1945
```

Поддерживаются задержки `30s`, `10m`, `2h`, `1d`. Если указать просто число, оно считается минутами.

Команда `/scheduled` показывает текущую очередь отложенных блоков. Очередь хранится в `scheduled_quizzes.json`, поэтому не пропадает после перезапуска бота.

### Фото

Если отправить фото с подписью в формате вопроса или блока, бот сначала опубликует фото, затем викторину. Если отправить фото без подписи, следующее текстовое сообщение будет опубликовано вместе с этим фото в пределах `PHOTO_TTL_SECONDS`.
